Квадраное уравнение

Квадратное уравнение — уравнение вида ax2 + bx + c = 0, где a, b, c — некоторые числа (a ≠ 0), x — неизвестное.

Числа a, b, c называются коэффициентами квадратного уравнения.

  • a называется первым коэффициентом;
  • b называется вторым коэффициентом;
  • c называется свободным членом.

Корни квадратного уравнения зависят от знака дискриминанта (D)

Введите коэффициенты a, b и c.

Если коэффициент не введен, он приравнивается к 0

Функция вида y = ax2 + bx + c
